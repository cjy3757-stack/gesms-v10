GESMS V10.5.2 가족구분 정상반영

실제 GESMS_군우명단.xlsx 검증 결과
- 군우: 94명
- 가족구분: 42가정
- 사용 컬럼: 가족구분, 성명, 생일구분, 생일, 직분, 전화번호, 소속1~4

수정 내용
- 실제 엑셀의 '가족구분' 열을 최우선으로 읽도록 수정
- A, B, C, ... AJ 등의 가족구분 값을 그대로 사용
- 같은 가족구분의 군우를 하나의 가족으로 묶음
- 기존처럼 군우마다 1~94의 자동 가족코드를 만들지 않음
- 가족구분 열을 찾지 못하면 잘못된 명단을 만들지 않고 오류 안내
- 총가정, 총교인, 평균 가족원 계산에 동일한 가족구분 데이터 사용
- 홈의 군우명단 반영 문구에 군우 수와 가정 수를 함께 표시
- 서비스워커 캐시 버전 갱신

정상 반영 예상
- 최신 군우명단 반영: 94명 · 42가정
- 가족명단에서 가족코드 A, B, C ... 형태로 표시
- 각 가족의 구성원 수가 엑셀 가족구분에 따라 표시
