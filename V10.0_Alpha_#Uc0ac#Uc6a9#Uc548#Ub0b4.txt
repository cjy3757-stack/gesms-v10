GunpoEunhye Smart Ministry System (GESMS) V10.0 Alpha
군포은혜영문 스마트 목양 시스템 V10.0

[Alpha 주요 변경]
1. GESMS 신규 브랜드명 및 원형 앱 로고 적용
2. 홈 화면 스마트 목양 대시보드 구조 적용
3. 전도(영혼구원) 대표 메뉴 신설
   - 전도현황
   - 기도대상자관리
   - 일만상상 프로젝트
4. 설정 / Excel 데이터관리 화면 신설
5. PWA manifest, 앱 아이콘, 서비스워커 V10.0 Alpha 갱신
6. 기존 V9.1.1의 교인, 출석, 통계, 돌봄, 재정, 공지, 일정 기능 유지

[주의]
Alpha는 새 구조와 메뉴를 검증하는 버전입니다. 기도대상자 Excel 자동연동과 전도 자동통계는 Beta에서 구현 예정입니다.
