GESMS V10.6 Full Edition
========================

이 압축파일은 GitHub 저장소 전체 덮어쓰기용 완성본입니다.

적용 방법
1. 압축을 풉니다.
2. 압축 안의 모든 파일을 GitHub 저장소 최상위에 업로드합니다.
3. 같은 이름의 파일은 Replace files로 덮어씁니다.
4. Commit changes를 누릅니다.
5. 약 1~3분 후 앱을 열고 Ctrl+F5로 새로고침합니다.
6. 휴대폰 설치형 앱은 완전히 종료한 뒤 다시 실행합니다.

확인 사항
- 상단 버전: V10.6
- 통계 메뉴 상단: 목양 통계 대시보드
- 연령대, 성별, 시니어, 다음세대, 가족 규모, 직분, 소속, 생일 그래프

주의
