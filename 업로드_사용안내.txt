GESMS V10.2.2 군우명단 Excel 자동연동

GitHub 저장소 최상위에 아래 이름으로 군우명단 파일을 올리세요.

GESMS_군우명단.xlsx

업데이트 방법
1. 엑셀에서 군우명단을 수정하고 저장합니다.
2. 파일명을 GESMS_군우명단.xlsx로 유지합니다.
3. GitHub 저장소 최상위의 기존 파일을 덮어씁니다.
4. 앱에서 강제 새로고침하거나 앱을 완전히 종료한 뒤 다시 실행합니다.

배포 시 교체할 파일
- index.html
- sw.js
- member-excel.js
- data.js (예비명단용, 기존 파일을 그대로 두어도 됨)

엑셀에서 자동 인식하는 주요 열
가족코드, 가족대표, 이름(또는 성명), 양력생일, 생일구분,
직분, 전화번호, 소속1, 소속2, 소속3, 소속4, 비고

주의
- 명단 Excel은 반드시 GitHub 저장소 최상위에 올립니다.
- jszip.min.js는 기존 저장소에 있어야 합니다.
- Excel을 읽지 못하면 기존 data.js 명단을 예비자료로 사용합니다.
