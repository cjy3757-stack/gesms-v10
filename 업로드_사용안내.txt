GESMS V10.4 안정형 완전판

핵심 수정
- 기존 V10.2 Release 2의 전역 이벤트 구조를 유지하여 출석 버튼 정상 작동
- init 함수만 비동기로 변경하여 군우명단 Excel을 먼저 읽은 뒤 화면 초기화
- APP_DATA 객체를 교체하지 않고 내부 데이터만 갱신하여 기존 기능과 충돌 방지
- 가족 규모 현황을 가족 구성원 수 기준으로 안정 계산
- 교회 내부용 한 줄 표시
- GitHub Excel/스크립트 캐시 방지
- Excel 실패 시 data.js 예비명단 사용
